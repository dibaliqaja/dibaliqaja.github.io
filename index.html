Tentang Kami

Tuban Creative Hub Dev

TCH Dev merupakan perusahaan di bidang teknologi yang berasal dari Tuban. Kami memberikan layanan servis dan jasa mulai dari mobile development, web development, dan design and multimedia. Selain itu, kami mengembangkan produk - produk teknologi yang dapat diimplementasikan untuk meningkatkan produktivitas dan layanan kerja. Kami mampu memberikan solusi teknologi yang tepat dan terbaik sesuai dengan kebutuhan client.

Layanan Kami

Kami Memberikan Layanan Teknologi yang Tepat dan Terbaik Bagi Anda

Mobile App Development

Kami menawarkan pembuatan mobile app development pada platform Android mulai dari aplikasi hiburan hingga aplikasi untuk bisnis yang sesuai dengan kebutuhan client baik untuk personal maupun perusahaan.

Web Development

Aplikasi berbasis web saat ini sangat dibutuhkan untuk semua pelaku bisnis dari UMKM sampai ke tingkat pemerintahan. Karena hal ini untuk mempermudah dan mempercepat suatu pekerjaan dan semua menjadi lebih transparan.

Design and Multimedia

Kreatif, Inovatif dan Komunikatif adalah bagian terpenting di dalam tim kami saat merancang semua produk desain untuk client. Baik desain grafis, video, dan animasi, sehingga produk yang dihasilkan bisa sepenuhnya sesuai target dan bisa menjadi ujung tombak saat melakukan promosi.

Tools and Technology

Tools dan Technology yang Kami Gunakan Untuk Membangun Project

NodeJS

NodeJS adalah platform software yang berjalan pada sisi server dan aplikasi jaringan. Kami menggunakan NodeJS untuk Backend dan Frontend dalam pengembangan aplikasi

Laravel

Laravel adalah Framework PHP Open Source Untuk Web Development. Kami menggunakan laravel untuk pengembangan aplikasi berbasis web seperti sistem informasi

Android Studio

Android Studio adalah tool yang digunakan untuk pengembangan aplikasi berbasis android. Kami membangaun dan merancang aplikasi android menggunakan tool ini.

Trello

Trello adalah board virtual untuk mengelola tugas. Kami menggunakan Trello untuk kolaborasi manajemen tugas - tugas dalam tim terkait proyek yang sedang kami kerjakan.

Git

Git adalah Version Control System pada proyek pengembangan software. Tim kami menggunakan Git untuk bekerja sama dalam proyek yang kami kerjakan

VueJS

VueJS adalah Framework Java Script Open Source Untuk Web Development. Kami menggunakan VueJS untuk pengembangan aplikasi frontend berbasis web

Adobe XD

Adobe XD adalah aplikasi yang kami gunakan untuk design UIUX dalam perancangan aplikasi berbasis web maupun mobile.

Adobe Premiere

Adobe Premiere adalah aplikasi yang kami gunakan untuk editing multimedia seperti video trailer, video sponsor, dan sebagainya.
